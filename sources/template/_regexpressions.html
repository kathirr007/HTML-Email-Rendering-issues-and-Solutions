Useful Regular Expressions

  1. Regular expressions to find all %LINK_ID in HTML creative
    %link_id=([^\s]*\d{8,})
  2. To find any html tag with closing tag from HTML

    <%TAG%\b[^>]*>(.*?)|<\/%TAG%>

    Here replace %TAG% with the tag which needs to be match.
